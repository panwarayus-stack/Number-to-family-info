Help & FAQ

Find answers to commonly asked questions about our API

How do I get started with the API?

To get started, simply sign up for an API key on our platform. Once you have your key, you can start making requests to our endpoints. We provide comprehensive documentation and code examples to help you integrate quickly.

What data can I retrieve with the API?

Our API allows you to retrieve comprehensive family information including family members, their relationships, addresses, verification status, and more. All data is sourced from reliable databases and presented in a structured format.

Is my data secure?

Yes, we take data security very seriously. All API requests are encrypted using HTTPS, and we follow strict privacy protocols. We never store sensitive personal information beyond what is necessary for the API functionality.

What is the pricing for the API?

We offer a tiered pricing model with a free tier for testing and development. For production use, we have several plans based on your usage needs. Contact our sales team for custom enterprise solutions.

How can I get support?

We offer multiple support channels including email, live chat, and a dedicated developer portal. Our support team is available during business hours to help with any technical issues or questions.
